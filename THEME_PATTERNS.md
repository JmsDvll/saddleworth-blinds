# Saddleworth Blinds Theme Patterns

## Page Categories & Their Themes

### 1. HOME PAGE (Special/Hero)
**Purpose**: Main landing, showcase everything, drive conversions
- **Hero**: Gradient background, large carousel, gold accent headings
- **Sections**: Alternating dark/darker backgrounds
- **Features**: Cards with hover effects
- **CTAs**: Multiple, prominent buttons
- **Spacing**: Large padding for impact

### 2. PRODUCT PAGES (Showcase)
**Purpose**: Detailed product information, specifications, gallery
- **Hero**: Dark background, product name with gold accent
- **Structure**:
  - Hero with title/description
  - Key features in cards
  - Product carousel 
  - Benefits section
  - Specifications/details
  - FAQ section
  - CTA section
- **Consistent Elements**:
  - Dark theme throughout
  - Gold accent headings
  - Card-based feature lists
  - Same spacing patterns

### 3. AREA PAGES (Local/Trust)
**Purpose**: Local SEO, build trust with area residents
- **Already Consistent**: Using createAreaPage template
- **Hero**: Gradient background with area name
- **Sections**: Property types, local expertise, service info
- **Visual**: Mix of cards and text sections

### 4. FORM PAGES (Action)
**Purpose**: Capture leads, contact information
- **Layout**: Split screen or centered form
- **Background**: Darker to focus on form
- **Form Styling**: Consistent input styles
- **Supporting Content**: Benefits, trust signals

### 5. UTILITY PAGES (Information)
**Purpose**: Legal, gallery, supporting content
- **Simple Layout**: Single column, readable
- **Minimal Styling**: Focus on content
- **Consistent Typography**: Same heading/text patterns

## Universal Theme Rules

### Colors
- **Primary Text**: White/Light gray on dark backgrounds
- **Accent**: Gold (#FFD700) for headings and CTAs
- **Backgrounds**: 
  - Dark (#1a1a1a)
  - Darker (#0f0f0f) 
  - Gradient (dark to darker with gold hints)

### Typography
- **Hero Headings**: 4xl mobile, 6xl desktop
- **Section Headings**: 3xl mobile, 4xl desktop  
- **Sub-headings**: xl to 2xl
- **Body Text**: base to lg

### Spacing
- **Section Padding**: "large" (64px/4rem)
- **Container Max Width**: 
  - Product pages: "large"
  - Area pages: "medium" 
  - Utility pages: "medium"
- **Stack Spacing**: "large" between major sections

### Components Usage
- **Cards**: For features, benefits, grouped content
- **Buttons**: Primary for main CTAs, Secondary for alternatives
- **Icons**: Gold accent in feature lists
- **Images**: Consistent aspect ratios per type
